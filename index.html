8:What s the difference between window vs document?

Ans: The major difference between window and document are window object represents the browser window or tab,
whereas the document object represents the HTML document within that window.
The window object provides access to the overall browser environment,
while the document object allows manipulation and interaction with the specific HTML document loaded in the browser.
